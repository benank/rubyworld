# Ruby World

[**Try it out here!**](https://rubyworld.benank.com)

Realtime online 2D game world. Built with Cloudflare Durable Objects & Pages, React, and Typescript. Inspired by Pok√©mon Ruby.

Uses:
 - Cloudflare Durable Objects + Cloudflare Workers + PartyKit for realtime communication via websockets
 - React / Typescript
 - shadcn/ui
 - Tailwindcss
 - Cloudflare Pages

## Developing

1. Install all packages with `npm install`
2. Run `npm run dev` to start the development server.

## Building

1. Run `npm run build` to generate static files in `/dist`.
